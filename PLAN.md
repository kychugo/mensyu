# 文樞（Mensyu）合併計劃 — 完整遊戲學習設計方案

> **狀態：待確認，尚未進行任何程式碼修改**
> 本文件整合現有六個版本（v1、v2、v3、mensyu、mensyu2、mensyu-tran）的所有優點，設計一套完整的文言文遊戲學習旅程。

---

## 一、現有版本盤點

| 版本 | 核心特色 | 保留元素 |
|------|---------|---------|
| **v1** | 基礎翻譯工具，點擊顯示釋義 | 翻譯 API 調用邏輯、字句逐一解釋 |
| **v2** | 浮動原文視窗、響應式佈局 | 浮動視窗、行動裝置適配 |
| **v3** | 雙作者（蘇軾/韓愈）、5關卡制、社交動態、問答測試 | 關卡架構、作者系統、社交互動 |
| **mensyu** | 三種模式（閱讀/測驗/遊戲），3D 打磚塊遊戲，閱讀紀錄 | 遊戲引擎、測驗模式、歷史紀錄 |
| **mensyu2** | 流暢的作者卡片與進度追蹤、個性化 AI 貼文 | 個性化 AI 回應、進度圓圈視覺化 |
| **mensyu-tran** | 輕量翻譯、浮動參考視窗 | 極簡翻譯入口 |

## 四、遊戲學習流程（逐關詳述）

### ▌關卡總覽

每位作者分 **5 個大關**，每大關包含 **3 個子階段**：

| 大關 | 難度 | 主題 |
|------|------|------|
| 第 1 關 | ⭐ 入門 | 認識作者背景、讀懂短句 |
| 第 2 關 | ⭐⭐ 初階 | 字詞解析、常見文言句式 |
| 第 3 關 | ⭐⭐⭐ 中階 | 段落翻譯、語境判斷 |
| 第 4 關 | ⭐⭐⭐⭐ 進階 | 全文翻譯、修辭手法辨識 |
| 第 5 關 | ⭐⭐⭐⭐⭐ 精通 | 自由文本挑戰、跨文本比較 |

---

```
子階段 A：「讀」— 閱讀模式

└─────────────────────────────────────────┘
            │ 完成閱讀 →
            ▼
子階段 B：「練」— 小遊戲選擇（三選一）
┌─────────────────────────────────────────┐

│                                         │
│  🎮 打磚塊（Breakout）                  │

│     • 球擊中磚塊前需選出正確釋義        │
│     • 生命值 ♥♥♥，答錯耗血             │
│                                         │
│  🃏 記憶配對牌                          │
│     • 文言字詞 ↔ 現代釋義 配對          │

            │ 任一遊戲完成 →
            ▼
```

## 五、「古人茶館」聊天系統設計

### 解鎖條件
- **完成第 1 關** → 解鎖與該作者的基礎問答（回應為引用原著語句）
- **完成第 3 關** → 解鎖深度對話（作者以當代個性回應生活問題）
- **完成第 5 關** → 解鎖自由創作模式（請作者品評使用者自寫文言文）

### 聊天模式

| 模式 | 說明 |
|------|------|
| **史實問答** | 詢問作者生平、歷史事件，AI 以第一人稱回應 |
| **作品探討** | 討論已學的具體篇目，作者解釋創作背景 |
| **古今對話** | 以現代話題提問，作者用古人視角回答 |
| **文章品評** | 使用者提交自創文言文，作者給予風格點評 |

### AI 人格設定（繼承自 mensyu2）
- **蘇軾**：樂觀豁達、好詠山水、幽默自嘲，擅以佛道思想解憂
- **韓愈**：剛直嚴謹、重道德倫常、言辭犀利，力主古文改革

### 社交動態廣場（繼承自 v3 / mensyu）
- 作者定時自動發表動態（預設 60 分鐘可調）
- 使用者可留言，作者 AI 回覆
- 動態按作者篩選（全部 / 蘇軾 / 韓愈）
- 顯示發文時間、按讚數

## 六、小遊戲詳細設計

### 🎮 遊戲一：文磚挑戰（Breakout，源自 mensyu）

| 項目 | 說明 |
|------|------|
| 來源版本 | mensyu 的 3D 打磚塊遊戲引擎 |
| 磚塊內容 | 每塊磚顯示一個文言字詞 |
| 答題機制 | 球接近磚塊時，底部出現 4 個翻譯選項，選對才能擊破磚塊 |
| 強化版 | 進階關卡：磚塊顯示現代詞，需選出對應的文言字 |
| 生命值 | ♥♥♥，答錯 / 球落底扣血 |
| 獎勵 | 完美通關（不扣血）額外加星 |

### 🃏 遊戲二：古今配對（Memory Match，新設計）

| 項目 | 說明 |
|------|------|
| 牌面設計 | 16 張牌（8 對），正面朝下 |
| 配對內容 | 文言字詞 ↔ 現代釋義；或句子 ↔ 翻譯 |
| 計時 | 限時 90 秒，越快越高分 |
| 難度遞進 | 初級 8 對字詞 → 中級 12 對含句子 → 高級 16 對含成語 |

### ✏️ 遊戲三：填詞對聯（Fill-in-the-Blank，新設計）

| 項目 | 說明 |
|------|------|
| 題目形式 | 顯示原文句子，遮蔽 1–3 個關鍵字 |
| 作答方式 | 從下方 4 個選項中拖曳或點選正確字詞 |
| 連鎖模式 | 連續答對 3 題觸發「連鎖獎勵」，加倍得分 |
| 進階玩法 | 精通關卡：無選項，直接輸入文字 |

## 七、進度與獎勵系統

### 進度視覺化（繼承自 mensyu2 進度圓圈）
- 每位作者顯示 5 個進度圓圈
- 已完成：實心彩色圓 ✅
- 進行中：半填充動畫圓
- 未解鎖：灰色空心圓 🔒

### 成就徽章系統
| 徽章 | 解鎖條件 |
|------|---------|
| 🔰 初入文場 | 完成任一作者第 1 關 |
| 📖 博覽古籍 | 閱讀 10 篇不同文本 |
| ⚔️ 磚場勇士 | 打磚塊遊戲完美通關 3 次 |
| 🏆 文樞大師 | 完成所有作者所有關卡 |
| 💬 古今對話 | 與每位作者各聊 10 條訊息 |
| ✍️ 以古喻今 | 提交自創文言文並獲作者品評 |

### 星級評分（每關最多 3 星）
- ⭐ 通過測驗（≥60 分）
- ⭐⭐ 高分通過（≥80 分）＋完成遊戲
- ⭐⭐⭐ 完美通關（滿分）＋遊戲不扣血

## 八、翻譯核心功能（整合所有版本）

### 翻譯頁面設計
- 左欄：輸入原文（支援任意文言文輸入）
- 右欄：翻譯輸出，包含：
  - 字詞逐一解析（點擊展開，源自 v1）
  - 句子完整翻譯（點擊展開）
  - **浮動原文參考視窗**（可調整大小，源自 v2）
- 三種查看方式切換：全展開 / 全隱藏 / 逐字測試模式

### 測試模式（源自 mensyu）
- 點擊任一字詞：出現 4 個選項
- 答對：字詞變綠色，顯示解析
- 答錯（最多 3 次）：字詞變紅色，強制顯示正確答案
- 完成整段：彈出得分報告

## 九、UI / UX 設計規範

### 視覺風格
- **配色**：以深沉的水墨藍（#2d4a7a）為主色，金黃（#d4a017）為點綴，仿古卷紙米白（#f5f0e8）為背景
- **字體**：正文使用楷體 / Noto Serif TC，標題使用書法風格字體
- **圖示**：毛筆筆觸風格 SVG 圖示

### 響應式斷點（繼承自各版本最佳實踐）
| 裝置 | 佈局 | 浮動視窗 |
|------|------|---------|
| 手機（<768px）| 單欄，頂部折疊 | 頂部可折疊面板 |
| 平板（768–1024px）| 雙欄 70/30 | 側邊浮動 |
| 桌面（>1024px）| 雙欄 60/40 | 可拖移 / 可縮放 |

### 無障礙設計（源自 mensyu2）
- 所有互動元素加入 `aria-label`
- 鍵盤可操作的遊戲控制
- 高對比度模式切換

- **文字生成**：`https://gen.pollinations.ai/v1/chat/completions`
- 翻譯請求、聊天對話、社交動態生成、文章品評均通過此端點
- 模型選用：依任務難度選 `openai` 或 `gemini` 模型
